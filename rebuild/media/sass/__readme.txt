Sass - разбивка
1)blocks - jade блоки, имеют префикс b- (повторяющиеся блоки)
2)components - элементы страниц (стиди кнопок, списков, декораторы)
3)template - базовый скелет header/wrap/footer, содержит глобальные контейнеры, сетки
4)utilities - подключаемые плагины

gulp-sprite
Для подключения спрайта юзается +sprite($sprite_name)

gulp icon-font
Для подключения иконки юзается +icon(iconName, before/after)
(если брать иконки из icomoon, то при сохранении нужно давать и размер ~200px
для корректного отображения потом в шрифтовом формате)
